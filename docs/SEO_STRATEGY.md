## Content Strategy

### Blog (Build-in-Public)
- Target: 1 post per shipping session via BIP pipeline
- Format: technical, lowercase, no hype
- Cross-linked with X posts via Typefully

### Documentation Pages
- Each tool-specific doc page targets "[tool name] MCP server" keywords
- Architecture page targets "AI coding tool architecture"
- Safety pipeline page targets "AI code security audit"

## Technical SEO

- Static HTML (11ty) — no JavaScript, instant load
- RSS feed at /feed.xml
- Semantic HTML with proper heading hierarchy
- No tracking scripts — privacy-first
- All pages < 100KB
- Dark/light mode via CSS (no JS toggle)
